Class: UISSMessage

UISSMessage

new UISSMessage(millis [, finalMessage])

Constructs a new UISSMessage object. Object is not ready to use after construction.

  • 1) var msg = new UISSMessage(millis, finalMessage)
  • 2) msg.setAction(idCount, msg.actions.x, [content]).allocate();
  • 3) use msg.getBytes() or msg.getByteArray() to send bytes to asgard
Parameters:
Name Type Argument Default Description
millis Number

of Date.now()
finalMessage Boolean <optional>
 true

flag intermediate messages with false

Members

actions :actions

Object containing supported actions. Data consists of action keys and the corresponding [0] ascii character indicating the action in the protocol, [1] a method to update the protocol structure and [2] a method to set the values of the updated structure. This object is used to define the action type with the UISSMessage#setAction method.
